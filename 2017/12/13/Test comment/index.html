Hexo 安装

一开始安装Hexo是在Mac上，折腾了一个晚上没有成功，查了一些资料包括给一些博主留言咨询，无奈第二天重新安装一遍，结果成功了。

所有的安装步骤都是一样的，后来分析应该是安装的路径导致的。
在Mac上安装成功之后，再在Windows上安装就轻松很多了，毕竟刚刚切换到Mac不到一个星期。

Next 安装

成功部署Hexo之后，就开始考虑换主题了，毕竟默认的主题还是没有那么的漂亮。一番搜索之后，发现Next这个美观简洁功能很强大的主题。

Next 安装也简单，直接根据READ.ME一步一步来。

Hypercomments 安装

Next 可以支持多种评论系统，多说 友言等等，一番了解后，还是决定使用Hypercomments.

使用Hypercomments也简单，不过需要翻墙。官网注册后，拿到Hypercomments_id，直接配置Next的config.yml就可以了。
不过在配置的过程中，几次没有生效。后来发现，先申请id，后安装Next，直接配置，直接就可以生效了。

Hypercomments 测试

文末留言就可以看到测试成功了，接下来就可以愉快的玩耍了。
